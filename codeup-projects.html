Quickly Navigating the World of Web Development

I'm currently learning the essential building blocks of the web. My coding adventure is fueled by a mix of structured learning at Codeup LLC's Java Web Developer Bootcamp and a personal commitment to understanding the nuances of each technology I encounter. From the static markup of HTML to the dynamic functions of JavaScript, my focus is on creating functional, user-centric web experiences.

This site is a practical application of CSS Flexbox, demonstrating the layout's potential even with a foundational understanding. While the design is not fully adaptive across all devices, employing a single media query has allowed me to explore and implement a basic responsive design. It's a clear example of how even beginner-level Flexbox skills can lead to creating structured and visually appealing web pages.
